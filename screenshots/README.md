# Screenshots Guide

To capture screenshots for submission, run the dashboard locally:

    streamlit run app.py

Then capture:
1. Home page (metrics overview)
2. Prediction form (filled in)
3. Prediction results (with recommendations)
4. EDA page (correlation heatmap view)
5. Analytics page (interactive Plotly charts)
6. Reports page

Save each as screenshot_01.png ... screenshot_06.png in this folder.
